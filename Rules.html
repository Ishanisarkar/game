The Game is a quiz test to check how the user is known with the traffic symbols. Level-1.

The Rules and Regulations of the game are as follows:
  • Only a single player can play the game at a time.
  • The user can play the levels sequentially, after clearing the level the next level will be unlocked.
  • The user will be given with an image of a traffic symbol, he/she needs to identify the sign board.
  • For each correct answer, 2 marks will be rewarded and for incorrent answer 0 marks will be deducted.
